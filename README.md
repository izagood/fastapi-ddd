# Fastapi Domain-Driven-Design(DDD) Example
An example of applying Domain-Driven-Design to FastAPI.


## Directory structure
This is a Domain-Driven-Design directory structure with Member examples.

```
├── fastapi_ddd
│   ├── application
│   │   └── member
│   ├── common
│   │   ├── config
│   │   └── exception
│   ├── domain
│   │   ├── custom_types
│   │   └── member
│   ├── infra
│   │   └── database
│   │       └── alembic
│   │           └── versions
│   └── presentation
│       └── rest
└── tests
```
